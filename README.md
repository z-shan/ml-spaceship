# React App for SpaceShip Inventory

## Prerequisites
nodejs  > 4

Developed on 7.1.0
Tested on 4.7.3 and 7.1.0

## Run
````
1. npm install
2. Gulp
````

## Components
* Container : App, InventoryPage
* Presentation : AboutPage, HomePage, InventoryList, SpaceShipModal
